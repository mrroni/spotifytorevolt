Spotify ve Revolt API erişim token'larınızı girin:

Client ID, Client Secret, Refresh Token ve Revolt Token Nasıl Elde Edilir?

Spotify Client ID ve Client Secret: Spotify Developer Dashboard'a gidin. Yeni bir uygulama oluşturun ve uygulamanızın ayarlarında Client ID ve Client Secret'ınızı göreceksiniz.

Spotify Refresh Token: Refresh Token'ı almak için, kullanıcıdan erişim izni almak için gerekli OAuth 2.0 akışını takip etmeniz gerekir. İlk erişim isteği sırasında refresh token'ı alabilirsiniz. Detaylar için Spotify API belgelerini inceleyin.

Revolt Session Token: Revolt hesabınıza giriş yaptıktan sonra, tarayıcı konsolunu açarak (Ctrl + Shift + I) Application kısmından IndexedDB'nin içinde token'ı görebilirsiniz. Gerekirse bir API istemcisi kullanarak token'ı alabilirsiniz.
